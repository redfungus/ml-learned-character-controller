# Unity Project Prototype

Unity demo for the quaternion orientation prediction model.

## Requirements

- Unity 6000.0.36f1 (Unity 6)

## How to Run

1. Open the project in Unity Hub
2. Open the scene `Assets/Scenes/RotationTest.unity`
3. Press Play to run the demo
   - Use the arrow keys or a Xbox controller trigger to move the planes
   - Press R or Y on the controller to reset
   - Press and hold space or A to input uniform data that matches the control input distribution of the data
4. The used model can be changed in the `Assets/OrientationModelSettings` ScriptableObject
